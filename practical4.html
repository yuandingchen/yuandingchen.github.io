Paragraphs

A paragraph (from the Greek paragraphos, “to write beside” or “written beside”) is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.

What is selector specificity?

Specificity is the algorithm used by browsers to determine the most relevant CSS declaration for an element. It calculates the weight of CSS selectors and thus determines which rule should be applied to the element. Specificity comes into play after considering the source and significance of the cascade.

How is specificity calculated?

The specificity algorithm assigns each selector a three-column value:

ID column: Include only ID selectors (e.g. #example). For each ID in the match selector, add 1-0-0 to the weight.

CLASS column: includes class selectors, attribute selectors (such as [type="radio"]) and pseudo-classes (such as: hover). For each class, attribute selector or pseudo-class, add 0-1-0 to the weight.

TYPE column: includes type selectors (such as p, h1, td) and pseudo-elements (such as::before, ::placeholder). For each type or pseudo-element, add 0-0-1 to the weight.

Universal selectors (*) and pseudo-classes: where() have values ​​0-0-0 but will still match elements without affecting specificity. Combinators (such as +, >, ~, " ", ||) do not add specificity weight.

What are complex selectors?

Complex selectors in CSS are a way to select elements based on a combination of conditions. They consist of one or more simple selectors or compound selectors separated by combinators. Combinators like descendant ( ), child (>), adjacent sibling (+), and general sibling (~) allow you to target elements based on their relationship to other elements in the DOM

What is the difference between pseudo-classes and pseudo-elements?

Pseudo-classes and pseudo-elements are both used in CSS to apply styles under certain conditions, but they target different things:

Pseudo-classes(:) are used to define the special state of an element. For example, :hoverwill apply a style when the user hovers over an element. They can also target elements based on their state or position in the document, like :first-child which targets the first child element of its parent.

Pseudo-elements (::) allow you to style specific parts of an element. For example,::beforecreates a pseudo-element that is the first child of the selected element, allowing you to insert content at the beginning of an element. Pseudo-elements can also style portions of content, like ::first-line or ::first-letter, which apply styles to the first line or first letter of a text block

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.
